# Smart-lab Парсер

Автоматический сбор и очистка новостей с финансового портала Smart-lab.ru.

## Что делает
- Парсит 50 последних постов с сайта
- Очищает от HTML-мусора (реклама, меню, скрипты)
- Сохраняет в текстовый файл
- Отправляет результат на почту

## Как запустить
1. Установи зависимости: `pip install -r requirements.txt`
2. Настрой почту в файле `smartlab_clean_parser.py` (укажи свой логин и пароль)
3. Запусти: `python smartlab_clean_parser.py`

## Автоматизация
Настроен ежедневный запуск через Планировщик задач Windows в 12:00.
